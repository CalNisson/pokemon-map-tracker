This tracker assumes you're using Pointcrow's Pokémon Emerald Warp Randomizer.

The map uses a color code of red (city), green (route) and blue (other locations). The progress tracker can be used to mark warps too.

Warp linking:
  • Left click on a warp to start linking. On this state, you can right click to cancel it, change location, click a mark to set it or click another warp to create a link.
  • Right click on any known warp to undo and mark it as unknown (if it's a link, it will mark the other end as unknown too).

Location changing:
  • Left click on the map to change current location.
  • Left click twice on a linked warp to change to the linked location.

Progress tracking:
  • Right click on any highlighted mark to cycle through all the maps with the mark in it.
  • Left click on the progress tracker to mark an objective as completed.
